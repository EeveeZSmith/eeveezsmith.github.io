How does CSS use color?

CSS can customize color in a variety of ways, including:

  • Color: Changes an element's color. For example,
    this text color has been set to Indigo.
  • Background-color: Changes the color of an object's background. The body of this page has been changed to Lavender Blush.
  • Border-color: Changes the border of an element, once specified, to a particular color. This element has a Lavender border.
