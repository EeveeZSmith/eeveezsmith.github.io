How does CSS use color?

CSS can customize color in a variety of ways, including:

  • Color: Changes an element's color. For example,
    this text color has been set to Indigo.
  • Background-color: Changes the color of an object's background. The body of this page has been changed to Lavender Blush.
  • Border-color: Changes the border of an element, once specified, to a particular color. This element has a Lavender border.

CSS accepts input for defining colors in several ways:

  • Named colors refer to a pre-programmed list of colors that work with CSS. The examples above use named colors.
  • Hexadecimal colors use a code made up of a pound sign followed by six letters and/or numbers. For example, Lavender would be #E6E6FA.
  • RGB specifies a combination of red, green, and blue values to make a color. Just like mixing paint! Number values are used to define how much of each color goes into the mixture.
  • Grayscale, similarly, can be rendered by defining values of black, white, and gray.
