How does a CSS media query work, and why is it useful?

A CSS media query allows a websiteâ€™s design to adjust itself to various properties of the specific device a user is browsing on. It does this by giving multiple coded layout options and allowing the browser to pick the one most appropriate for the properties of the device, such as its size and whether it uses a touchscreen. This is called Responsive Design, and it is essential for making websites display and function appropriately for the wide variety of devices now used to access the internet.

Examples of Media Features Supported by Media Queries:

  • Orientation = Adjusts the website layout for landscape or portrait mode
  • Hover = If a mouse is being used, allows the website to react when the mouse icon hovers over a certain object
  • Video Resolution = Determines what level of video quality should be displayed
  • Prefers Color Scheme = Tells the website top display a light or dark color scheme (if options are available) based on user settings
